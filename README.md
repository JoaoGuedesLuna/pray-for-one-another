<h2>📖 About the Project</h2>
<p align="justify">
  <strong>Pray for One Another</strong> was inspired by a small prayer group I shared with my friends, called <em>"Oração pela Juventude"</em> ("Prayer for the Youth"). Every Monday night, we would gather to read the Bible and pray together. At the end of each meeting, each person would pray specifically for another member. Wanting to bring a touch of technology into this meaningful routine, I had the idea to create this program to randomly assign prayer pairs — a simple solution to make our tradition easier and even more engaging.
</p>

<h2>🚀 How to Run</h2>
<h3>Using an IDE (e.g., IntelliJ, Eclipse)</h3>
<ol>
  <li>Clone or download this repository.</li>
  <li>Open the project in your Java IDE.</li>
  <li>Run the <code>dev.guedes.prayforoneanother.PrayForOneAnotherApplication</code> class.</li>
</ol>

<h3>Using the Executable JAR</h3>
<ol>
  <li>Navigate to the <code>/dist</code> folder.</li>
  <li>Double-click the <code>pray-for-one-another-1.0.jar</code> file or run it via terminal:<br>
        <code>java -jar .\dist\pray-for-one-another-1.0.jar</code>
  </li>
</ol>
